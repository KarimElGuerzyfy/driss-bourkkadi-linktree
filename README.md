# Driss Bourkkadi — Link Tree

A sleek, responsive Link Tree page built for Driss Bourkkadi, a multidisciplinary designer. The page serves as a single hub for all his social media and professional profile links.

## Overview

The page features a modern glassmorphism design on desktop, with a full-body background photo that bleeds into the foreground for a dynamic visual effect. On smaller screens, the layout adapts cleanly — the glass effect fades out, borders disappear, and the content centers itself to fit any screen size comfortably.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Poppins (Google Fonts)

## Features

- Glassmorphism card design (desktop only)
- Background photo with drop shadow bleeding into the foreground
- Fully responsive — desktop, tablet, and mobile
- Desktop layout capped at 1440 x 1024px
- Profile section with avatar, name, and bio
- Social links: Facebook, Béhance, WhatsApp, LinkedIn, Instagram
- Instagram button with custom brand gradient
- Component-based architecture
