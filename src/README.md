# Driss Bourkkadi — Link Tree

A clean, modern Link Tree page built for Driss Bourkkadi as a personal profile link hub.

## About

This is my first commission work as a developer/designer. The project is currently in development and will be transferred and deployed to the client upon completion.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS

## Features

- Glassmorphism card design
- Responsive layout (PC optimized at 1440px)
- Social media links: Facebook, Béhance, WhatsApp, LinkedIn, Instagram
- Custom background photo with drop shadow effect
- Poppins font
